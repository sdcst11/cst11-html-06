Create a Template

When you are working with creating a navigation system, it is best to have your links already created so you can create a basic template that will be used for all of your pages. While there have been many methods that have been used to create and organize a menu system, the current most popular method is to use an unordered list using the <ul></ul> tags. You will also need to include individual <li></li> tags for each menu item. Consider the example below:

Note that this menu system shows the basic hyperlink. It has the blue text that is underlined which is the old way of showing that a section of text is a hyperlink, and there is a bulletpoint at the front of each list item that is created by the <li></li>.
